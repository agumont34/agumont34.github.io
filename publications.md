<h3 style="margin-top:2em;"> Contribution to selected software prototypes </h3>

*   **Holistic embedded system design**. [MAGPIE](http://www.lirmm.fr/continuum-project/pages/magpie.html) is a collaborative framework that targets an easy design evaluation for multicore systems, based on gem5, McPAT and NVSim tools. It is made available to research community as open-source code to which any contribution is welcome for improvements and/or extention.

*   **Multicore system simulation**. [McSim](https://github.com/DreamCloud-Project) is a set of simulators dedicated to the design of embedded applications described using the AMALTHEA application model on top of manycore architectures.

*   **Multiprocessor system-on-chip co-design**. [Gaspard2](http://www.gaspard2.org) is a codesign model-driven environment for high-performance embedded systems.

*   **Formal modeling of safety-critical apps**. I defined a library of polychronous component models for the design of avionic applications in [Polychrony](http://polychrony.inria.fr), based on the avionic APEX-ARINC 653 standard. A vintage presentation of this library is available [here](http://hal.inria.fr/docs/00/07/19/07/PDF/RR-4678.pdf).
